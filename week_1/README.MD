# Week 1 HTML Assignment

## Project Title
Multi-Page HTML Website

---

## Objective
The objective of this project is to reinforce HTML fundamentals by creating a structured multi-page website that includes document structure, navigation links, multimedia elements, and organized content.

---

## Pages Included
This website contains four pages:

1. Home Page
2. About Page
3. Contact Page
4. External Links Page

---

## Features Implemented

### Proper HTML Structure
Each webpage includes:
- `<!DOCTYPE html>`
- `<html>`
- `<head>`
- `<body>`

### Meta Tags
The pages include:
- Character encoding meta tag
- Viewport meta tag
- Description meta tag

### Navigation
The website uses internal navigation links between pages:
- Home
- About
- Contact
- External Links

Navigation links are arranged vertically.

### Multimedia
Two images were added:
- One local image
- One online image from Unsplash

### Lists
The website contains:
- Ordered lists
- Unordered lists

### External Links
External websites are included for learning web development.
