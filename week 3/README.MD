# Week 03 Lab

## Objective
To learn how to use:
-To create a well-structured HTML webpage with multiple sections.
-To apply CSS styling using external, internal, and inline CSS methods.
-To demonstrate the use of different CSS selectors such as element, class, ID, group, descendant, and pseudo-class selectors.
-To design a simple and user-friendly navigation menu using anchor links.
-To improve webpage appearance using colors, typography, spacing, and layout styling.

-## The Website Includes

The website includes the following sections and features:

 - **Header Section**

   * Student name
   * Title or profession

 - **Navigation Menu**

   * Links that navigate to different sections of the webpage using anchor links

  - **About Section**

   * A short paragraph introducing the student
   * A profile image

  -**Skills Section**

   * A list of skills such as HTML, CSS, networking, or GIS

 - **Daily Routine Section**

   * An ordered list showing daily activities

 - **CSS Styling Features**

   * External CSS for general page styling
   * Internal CSS for navigation and headings
   * Inline CSS for a few specific elements

  - **CSS Selectors Demonstrated**

   * Element selector
   * Class selector
   * ID selector
   * Group selector
   * Descendant selector
   * Pseudo-class selector (`:hover`)
